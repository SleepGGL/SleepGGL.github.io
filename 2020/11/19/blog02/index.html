快速幂

本文最后更新于：5 天前

简述：通过二进制来优化加速求幂的过程，让时间复杂度达到log级别，ACM里面经常会用到的一个小技巧

快速幂

  • 算法思想:

例如计算$a^b$这样一个数，我们指数b以转换二进制的形式进行分解，将其写成二进制中每一位乘上该位的权重(从右往左，第i位的权为$2^{i-1}$)

例如：$a^{13}$ = $a_2^0$+$2^2$+$2^3$ = $a_2^0$*$a_2^2$*$a_2^3$

  • 时间复杂度

首先可以看到，循环的终止条件为指数e为0，且每次循环e都会右移一位，而自然数N的二进制长度为$log_2 N$，因此这个循环至多遍历$log_2 N$次。即它的时间复杂度为O($logN$)。

  • 快速幂代码
ll pow(ll x, ll y)  //位运算
{
    ll res = 1;
    while(y) {
        if (y&1)  res *= x ;   //res才是最终我们要的结果.
        x *= x ;     // 让x的次幂以2的倍数增长
        y=y>>1;     // 对指数进行移位
    }
    return res;
}
  • 快速幂取模代码
ll pow_mod(ll a ,ll b)
{
    ll ans = 1 ;
    ll base = a % mod;
    while(b>0)
    {
          if(b&1!=0)
            ans = (ans *base)%mod;
        base = (base*base)%mod;
        b >>= 1;
    }
    return ans;
}

struct node {
    int mat[15][15];    //定义矩阵
}x,y;

node mul(node x,node y){    //矩阵乘法
    node tmp;
    for(int i=0;i<len;i++){
        for(int j=0;j<len;j++){
            tmp.mat [i][j]=0;
            for(int k=0;k<len;k++){
                tmp.mat [i][j]+=(x.mat [i][k]*y.mat [k][j])%mod;
            }
            tmp.mat [i][j]=tmp.mat[i][j]%mod;
        }
    }
    return tmp;
}

node matpow(node x,node y,int num){//矩阵快速幂
    while(num){
        if(num&1){
            y=mul(y,x);
        }
        x=mul(x,x);
        num=num>>1;
    }
    return y;
}
